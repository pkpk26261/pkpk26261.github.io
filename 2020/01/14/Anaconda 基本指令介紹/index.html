Anaconda 基本指令介紹

發表於 2020-01-14 更新於 2020-04-29 分類於 Python 程式語言學習
文章字數： 922 所需閱讀時間 ≈ 1 分鐘

需要者，請自行閱讀此文章。

⬇⬇⬇文章開始⬇⬇⬇

請至 Anaconda 官網 下載安裝應用程式。
 

Anaconda 管理模組　

  1. 模組安裝
    pip為單一模組安裝， conda會幫你安裝該包，也會幫忙安裝其他所需模組。
  2. anaconda 升級
  3. pip 升級

安裝及更新conda

  1. 查看conda版本
  2. conda 更新

建立虛擬環境

  1. 查看目前有哪些環境
  2. 建立名為虛擬環境(name為自定義名稱)，並且安裝為python3.6版本。
  3. 啟動環境(name為要啟動的環境名稱)
  4. 離開環境
  5. 刪除環境(name為該環境名稱)

使用方式優缺點

從命令列執行 Python 互動式Shell

C:\Users\user>python ，即時看到結果，難以編修程式。

從檔案執行Python

C:\Users\user>python test.py，容易編修程式，無法即時看到結果。

Jupyter Notebook 編輯器

啟動Jupyter Notebook編輯器的方式有兩種，

  1. 在Anaconda Prompt下打入jupyter notebook
  2. 點選桌面的應用Jupyter Notebook

常用熱鍵

  1. Shift + Enter 執行再新增一列
  2. Ctrl + Enter 執行目前列
  3. Esc + a 在上方插入一列
  4. Esc + b 在下方插入一列
  5. Esc + dd 刪除一列
  6. Esc + h 顯示快速鍵列表
